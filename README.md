# Система автоматизации начисления заработной платы

**Краткое описание:**  
Проект представляет собой информационную систему для автоматизации процесса начисления заработной платы сотрудников организации. Система обеспечивает учет рабочего времени, расчет окладов, надбавок, премий, удержаний и налогов, а также формирование расчетных листов и отчетов для руководства.

**Основные функции:**  
- Ведение базы данных сотрудников  
- Учет рабочего времени  
- Автоматический расчет заработной платы  
- Расчет налогов и удержаний  
- Формирование расчетных листов и отчетов

**Цель проекта:**  
Снизить трудозатраты и количество ошибок при начислении заработной платы, повысить точность и оперативность формирования отчетной документации.

**Технологии:**  
- Backend: C#
- База данных: MSSQL 2021
- Frontend: Desktop интерфейс

**Статус проекта:**  
В процессе разработки в рамках учебной практики.
